Este repositório contém o desenvolvimento de um sistema de gerenciamento para a Biblioteca Padre Jaime Diniz, da Escola de Música da UFRN. O projeto utiliza PostgreSQL como sistema gerenciador de banco de dados e uma aplicação desktop em Python para o acesso dos usuários, estou trabalhando aos poucos em cada parte dele, mas todos os passos estão sendo comentados em um diário que em breve vou disponibilizar como um PDF, com a curva do meu aprendizado e as mudanças feitas e seus motivos.

A interface gráfica está sendo desenvolvida com o auxílio do Figma, integrando recursos da biblioteca Tkinter e ferramentas externas como o Tkdesign. A conexão com o banco de dados é feita por meio da biblioteca psycopg2, ideal para o Postgree.

Nesse repositório você vai encontrar:

A documentação completa do sistema,

Tutoriais de uso e instalação,

Código-fonte do projeto,

Histórico de desenvolvimento e atualizações (como esperado do git).
